# Dynamic programming - Longest Common Subsequence
### Problem
Find the longest common subsequence of two input strings `s`, and `t`
###  Input
Two strings `s`, and `t`
### Output
Return the longest common subsequence of two string `s`, and `t`
### Runtime
`O(n^2)`
### Note
+   `lcs.js` implement without closure.
+   `lcs_closure.js` implement using closure.
